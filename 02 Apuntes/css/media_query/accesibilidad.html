Accesibilidad

La web se adapta a las Pautas de Contenido Accesible en Web 1.0, nivel Doble A. Las páginas han sido creadas según la Recomendación XHTML 1.0 del W3C para el desarrollo web con marcado semántico y estructurado.

La capa de presentación se elaborará bajo la Recomendación del W3C sobre las Hojas de Estilo en Cascada, CSS3.

En el caso de que el navegador o dispositivo de navegación no sea compatible con hojas de estilo, el contenido de la web seguirá siendo totalmente legible gracias a su marcado estructural.

Todas las imágenes contendrán una breve descripción, a través de “alt”, “longdesc” o “title”.

Teclas de acceso

Los atajos de teclado, o “Access key”, son combinaciones de teclas que permiten acceder a los contenidos de la página web sin utilizar el ratón.

La forma de activar los atajos de teclado depende del sistema operativo y del navegador utilizado.

Las secuencias de teclas más comunes son:

En sistemas Windows, con navegadores Internet Explorer: ALT+ tecla acceso+ ENTER

En sistemas Windows, con Firefox 2.0 y 3.0: ALT+ SHIFT+ tecla acceso

En el sistema MAC, se debe teclear la secuencia: CTRL+ tecla acceso

En la web estarán disponibles las siguientes teclas de acceso:
